Editor de texto Atom y sincronización de su configuración en la nube

noviembre 15, 2015julio 4, 2016 / davr59

Hace algún tiempo recomendé editores de texto como Notepad++ y Brackets. Estos siguen siendo editores muy buenos pero ahora estoy utilizando uno diferente: Atom.

Es un editor de código abierto, moderno, flexible, con muchos temas y paquetes para extender sus funcionalidades. ¡Les recomiendo probarlo! Lo mejor: está para Windows, Mac y Linux ;-).

Un paquete que me llamó la atención es sync-settings. Este paquete permite a través de GitHub y Gist sincronizar la configuración y paquetes en diferentes instancias de Atom (en mi caso, Atom en Linux y Atom en Windows). Es muy fácil de configurar y me ha funcionado perfectamente

  1. Lo pueden encontrar dentro de Atom: Settings -> Install -> En el search poner sync-settings
  2. Luego sólo se ocupa configurarlo dentro de Atom: Settings -> Packages-> Seleccionar Settings de sync-settings
  3. Es necesario tener una cuenta de GitHub. En GitHub se debe crear un Personal Access Token y copiar ese token en la configuración anterior. Además se ocupar crear un Gist y copiar su ID (es el GUID en la dirección de URL) también en la configuracion anterior. ¡Y listo!
  4. Para utilizarlo en Atom: Packages-> Synchronize Settings -> Se puede seleccionar Backup o Restore

Me cuentan que les parece y cualquier consulta estoy a la orden. ¡Feliz semana!
